## Featured Public Project

### RNA Structure Visualizer

A browser-based workbench for inspecting RNA secondary structures, converting structure formats, comparing predictions, and exporting clean scientific figures.

**Live site:** https://rna-structure-visualizer.vercel.app

RNA Structure Visualizer provides a public-facing interface for practical RNA secondary structure analysis. It supports structure parsing, validation, visualization, format conversion, prediction comparison, and figure export in a browser-based workflow.

**Highlights**

* Parse and validate CT, BPSEQ, FASTA + dot-bracket, and plain sequence + dot-bracket inputs
* Visualize RNA secondary structures using circle and linear arc layouts
* Inspect pseudoknot-aware base-pairing patterns
* Convert structures between DBN, CT, BPSEQ, pair CSV, and empty CT template formats
* Compare predicted structures against references using exact or ¬Ī1 slip scoring
* Report PPV, sensitivity, F1, base-pair distance, and pair-level TP/FP/FN annotations
* Export clean scientific figures as SVG, PNG, or PDF
* Built with React, TypeScript, Vite, and deployed on Vercel
